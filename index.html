Timestamp microservice

This is an API project for FreeCodeCamp with the following functionality (mostly copied from the FreeCodeCamp site):

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp (in seconds) or a natural language date (example: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a valid date or Unix timestamp, it returns null for those properties.

Example usage:

/December%2014,%202015


/1450137600

Example output:

{unix: 1450137600, natural: "December 14, 2015"}
